## Getting started

**Prerequisites:**

- [NodeJS](https://nodejs.org/en/)
- git
- Access to [the Github repo](https://github.com/coderschool/coder-labs)
- Understanding basic [ReactJS](https://reactjs.org/)
- Understanding basic [MDX](https://mdxjs.com/)

**Clone the project:**

```bash focus=1
git clone https://github.com/coderschool/coder-labs.git
```

<CH.Section>

**Start the project:**

```bash
cd coder-labs
npm install
npm run dev
```

- [Move to the project folder](focus://1).
- [Install libraries and dependencies](focus://2), you only need to run this command once.
- [Start the development server](focus://3).

</CH.Section>

**Open <a href="http://localhost:3000/demo/coder-labs" target="_blank">localhost:3000</a>**
